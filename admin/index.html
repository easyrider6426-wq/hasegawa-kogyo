📷 画像の追加方法 / 手動保存モード
  1. 追加したい画像ファイルをパソコンの 画像素材/ フォルダへコピーします。
  2. 下の「新規追加」フォームの「画像ファイル名」欄に 画像素材/ファイル名.jpg の形式で入力します。
  3. 事例を保存し、生成された work-data.js をダウンロードして元のファイルと置き換えます。
  4. Cloudflare Pages へ再デプロイ（deploy.command を実行）することでWebサイトに反映されます。

⚙ GitHub APIを設定 すると、ボタン一つで自動デプロイできるようになります。

⚙ GitHub API 設定

GitHub Personal Access Token（PAT）を設定すると、「GitHubへ保存」ボタン一つでサイトが自動更新されます。
トークンはこのブラウザの LocalStorage にのみ保存され、外部に送信されません。

トークンの取得方法：
  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 「Generate new token (classic)」をクリック
  3. Note に任意の名前を入力し、repo（Full control of private repositories）にチェック
  4. 「Generate token」をクリックしてコピー
